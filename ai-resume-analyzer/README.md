# AI Resume Analyzer

Python project that analyzes a resume against a job description and identifies matching skills, missing keywords, and improvement areas.

## Project Overview

This project helps job seekers compare their resume to a job posting. It extracts resume text, checks for important keywords, calculates a match score, and generates suggestions for improvement.

## Technologies

- Python
- PDF Text Extraction
- Natural Language Processing
- Keyword Matching
- Data Analysis
- Reporting

## Features

- Extract text from resume files
- Compare resume content with job descriptions
- Identify matching skills
- Identify missing keywords
- Generate a resume match score
- Create improvement suggestions
