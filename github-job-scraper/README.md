# GitHub Job Scraper

Python project that collects software engineering job postings and saves them in a structured format for analysis.

## Project Overview

This project uses Python to collect job posting data, store it in a JSON file, and prepare it for further analysis. It demonstrates API usage, JSON processing, automation, and file handling.

## Skills Demonstrated

- Python
- APIs
- JSON Processing
- Data Collection
- File Handling
- Automation

## Project Goal

Build a tool that automatically gathers job postings and stores them in a structured format for further analysis.

## Project Files

- `scraper.py` — collects job posting data
- `jobs.json` — stores collected job data
- `requirements.txt` — lists required Python packages
- `README.md` — explains the project

## How to Run

```bash
pip install -r requirements.txt
python scraper.py
